📘 Potato Leaf Disease Classification (KNN)
📖 Deskripsi

Proyek ini bertujuan untuk mengklasifikasikan daun kentang menjadi:

Healthy
Early Blight
Late Blight

menggunakan metode K-Nearest Neighbors (KNN) berdasarkan fitur warna dan tekstur citra.

📊 Dataset
100 citra daun kentang
Sumber: Kaggle

📁 Dataset (Google Drive):
👉 https://drive.google.com/drive/folders/1f6kIoWCbsprKAdfPmzpNeNj7QlGNH7S6?usp=drive_link

⚙️ Metode
Ekstraksi fitur:
RGB & HSV
Tekstur GLCM
Split data: 70% training, 30% testing
Normalisasi: StandardScaler
Model: KNN (k=5)
📈 Hasil
Akurasi: 63.33%
